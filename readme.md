## Setup

1. Clone this repo
2. Install dependencies:
   pip install openai anthropic pymupdf
3. Set your API keys:
   export OPENAI_API_KEY="your-key-here"
   export ANTHROPIC_API_KEY="your-key-here"
4. Run any script:
   python week1/chatbot.py
